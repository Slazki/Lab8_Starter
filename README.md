## Graceful Degradation and Service Workers

Graceful degradation means the app should still provide a useful experience when newer browser features or network access are not available. Service workers support that goal by saving the files and responses the app needs in the browser cache, then serving those cached resources when the network is slow or offline. In this lab, the recipe app starts with normal fetch requests, but the service worker lets it keep working after those resources have been cached.
